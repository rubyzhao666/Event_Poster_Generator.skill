# 活动海报一键生成器

根据活动信息自动生成完整海报，智能判断活动类型并匹配视觉风格。

## 🎯 功能特点

- **活动类型智能识别**：讲座/沙龙/培训/比赛/发布会/团建
- **视觉风格自动匹配**：根据活动类型选择对应风格和配色
- **完整信息布局**：主视觉+亮点+时间地点+报名引导
- **高转化设计**：针对吸引报名优化

## 📥 输入

- 活动名称
- 活动时间、地点
- 核心亮点/嘉宾
- 目标人群
- 报名方式
- 品牌风格（可选）

## 📤 输出

- 竖版活动海报（3:4比例）
- 6个核心模块（主视觉、亮点、时间地点、目标人群、报名指引、主办方）
- 报名二维码区域预留

## 🚀 使用示例

**输入：**

> 活动名称：AI Agent开发实战工作坊
> 时间：2026年4月20日 14:00-17:00
> 地点：线上Zoom会议
> 亮点：从0到1搭建你的第一个Agent
> 目标人群：AI爱好者、开发者
> 报名方式：扫码入群

**输出：**

- 主视觉区：科技感设计，Agent概念可视化
- 亮点呈现：实战导向、零基础友好
- 时间地点：清晰醒目
- 报名引导：扫码区域突出
